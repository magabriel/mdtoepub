# Glossary

Active recall
:   A study strategy that involves retrieving information from memory without looking at the source material. Also known as the "testing effect".

Blocked practice
:   Practising a single type of problem or skill in a concentrated block, without mixing in other types. Contrasted with interleaved practice.

Cognitive load
:   The total amount of mental effort being used in working memory. High cognitive load impairs learning.

Deliberate practice
:   Purposeful, structured practice designed to improve specific aspects of performance, typically with feedback from a knowledgeable guide.

Elaboration
:   The process of explaining new information in your own words and connecting it to what you already know.

Encoding
:   The process of converting information into a form that can be stored in memory.

Forgetting curve
:   A graph showing the decline of memory retention over time, first described by Hermann Ebbinghaus in 1885.

Growth mindset
:   The belief that intelligence and abilities can be developed through effort, practice, and learning.

Interleaved practice
:   Mixing different types of problems or skills within a single study session, as opposed to blocked practice.

Long-term memory
:   The memory system responsible for storing information indefinitely, with a virtually unlimited capacity.

Memory palace
:   A mnemonic technique that involves associating information with specific locations in a familiar place.

Multitasking
:   The attempt to perform multiple tasks simultaneously. Research shows that what is commonly called multitasking is actually rapid task-switching, which reduces performance.

Pomodoro technique
:   A time management method that uses timed intervals (typically 25 minutes) of focused work followed by short breaks.

Retrieval practice
:   See *Active recall*.

Spaced repetition
:   A study technique that involves reviewing material at increasing intervals over time, exploiting the spacing effect.

Working memory
:   The memory system responsible for temporarily holding and manipulating information during cognitive tasks. Capacity is limited to approximately 7 +/- 2 elements.

Zone of Proximal Development (ZPD)
:   A concept from Vygotsky describing the gap between what a learner can do independently and what they can achieve with guidance.

**Note:** Terms in this glossary are defined as they are used in the context of this book. For broader or more technical definitions, consult the references listed in the Appendix.
